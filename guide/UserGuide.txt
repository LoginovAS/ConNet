0. Содержимое архива:
 - директория entity
    Содержит отдельную копию классов сущностей.
 - директория sql
    Содержит sql-скрипты для создания, заполнения тестовыми данными и удаления таблиц (см. п. 2.1).
 - директория source
    Содержит полный исходный код
 - connet.war
    Сборка приложения предназначенная для развёртывания на сервере приложений.

1. Платформа.
При необходимости произвести установку и минимально необходимую настройку согласно документации соответствующего продукта.
 - Glassfish 4.1
 - Java SE 8u121
 - MariaDB Server v.10.1.21

 2. Установка.

 2.1. База данных.

 - На сервере MariaDB (Mysql) создать новую базу данных:
 Пример:
 MariaDB [(none)]> create database connet;

 - Создать нового пользователя базы данных и выдать ему полные права на созданную базу.
 Пример:

 MariaDB [(none)]> grant all privileges on connet.* to connet@localhost identified by 'connet';

 - Зайти под созданным пользователем в базу и выполнить скрипт connet_test_schema.sql.
 Пример:
 MariaDB [connet]> source connet_test_schema.sql

 При наличии ошибок проверить текущую базу, к которой подключен пользователь, а также права на текущую базу.

 - Залить в базу тестовые данные. Для этого выполнить скрипт connet_test_data.sql.
 Пример:
 MariaDB [connet]> source connet_test_data.sql

 - При необходимости очистки тестовых данных запустить скрипт connet_drop_test_schema.sql

 2.2. Сервер приложений.

 - Добавить в папку lib рабочего домена glassfish mysql-connector-java.jar из архива либо скачать версию 5.1.39, если mysql connector отсутствует в домене (приложение тестировалось с этой версией библиотеки).
 - В консоли администратора glassfish создать connection pool для подключения к созданной базе данных
 - В консоли администратора glassfish создать DataSource с названием "jdbc/connet_ds". DS должен смотреть на connection pool из предыдущего пункта.
 - В консоли администратора в разделе Applications произвести деплой приложения используя war из архива (оставить параметры по-умолчанию).
 - По окончании деплоя запустить приложение нажав ссылку Launch.
